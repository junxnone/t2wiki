# Blender基础建模入门

## 简介

Blender是一款功能强大的开源3D创作软件，本文介绍Blender的基础建模概念和操作。

## 基本操作

### 导航视图

- **鼠标中键拖拽**：旋转视图
- **Shift + 鼠标中键**：平移视图
- **鼠标滚轮**：缩放视图
- **小键盘数字键**：切换正交视图

### 选择与变换

- **G键**：移动（Grab）
- **R键**：旋转（Rotate）
- **S键**：缩放（Scale）
- **Tab键**：切换编辑模式

## 基础建模工具

### 挤出（Extrude）

挤出是最常用的建模工具之一，可以从现有几何体创建新的面。

```
快捷键：E
操作：选中面 → 按E → 移动鼠标 → 左键确认
```

### 循环切割（Loop Cut）

用于在模型上添加边循环，细分模型。

```
快捷键：Ctrl + R
操作：悬停在边上 → Ctrl+R → 滚轮调整数量 → 左键确认
```

### 倒角（Bevel）

创建圆角效果，让边缘更加平滑。

```
快捷键：Ctrl + B（边倒角）、Ctrl + Shift + B（顶点倒角）
```

## 修改器

### 细分曲面（Subdivision Surface）

最常用的修改器，可以让模型更加平滑。

- 添加位置：修改器属性 → 添加修改器 → 细分曲面
- 细分级别：通常使用2-3级

### 镜像（Mirror）

对称建模的利器。

- 应用场景：人体、对称物体
- 轴向选择：X/Y/Z轴

## 实战练习：制作简单杯子

1. 添加圆柱体
2. 删除顶面
3. 选择底部边循环，使用内插面（I键）
4. 挤出底部
5. 添加细分曲面修改器
6. 调整形状

## 小技巧

- 使用**Shift+A**快速添加物体
- 按住**Shift**可以进行多选
- **Alt+点击**可以选择边循环或面循环
- **Ctrl+Z**撤销操作

## 总结

掌握这些基础操作后，你就可以开始创建简单的3D模型了。建议多练习，熟能生巧。
