## 核心区别

| 特性 | 实时渲染 | 离线渲染 |
|------|---------|---------|
| **速度** | 30-60+ FPS | 秒到小时/帧 |
| **质量** | 中高 | 照片级 |
| **交互性** | 实时交互 | 无交互 |
| **应用** | 游戏、VR、可视化 | 电影、动画 |
| **硬件** | GPU | CPU/GPU集群 |

## 实时渲染

### 特点

- ⚡ 必须保持高帧率（通常≥30 FPS）
- 🎮 用户可以实时交互
- 🔧 使用光栅化为主的技术
- 💡 近似和简化的光照模型

### 渲染管线

```
顶点处理 → 光栅化 → 片段着色 → 输出合并
```

### 常用技术

- **光栅化**：将3D几何转换为2D像素
- **延迟渲染**：分离几何和光照
- **实时阴影**：Shadow Maps
- **屏幕空间效果**：SSAO, SSR
- **光照贴图**：预烘焙光照

### 优化策略

- LOD系统
- 遮挡剔除
- 视锥剔除
- GPU实例化
- Draw Call批处理

### 主要引擎

- Unity
- Unreal Engine
- Godot
- Three.js

## 离线渲染

### 特点

- 🎨 追求最高质量
- ⏱️ 可以花费任意时间渲染一帧
- 📐 使用物理精确的算法
- 💎 完整的全局光照

### 渲染方法

#### 1. 路径追踪（Path Tracing）

模拟光线在场景中的真实传播。

**优点**：
- 物理准确
- 自然的软阴影
- 真实的间接光照

**缺点**：
- 计算密集
- 需要大量采样

#### 2. 光子映射（Photon Mapping）

从光源发射光子，记录它们的路径。

**适用**：
- 焦散效果
- 体积光
- 复杂光照场景

#### 3. 双向路径追踪（BDPT）

同时从相机和光源追踪光线。

**优势**：
- 处理复杂光路
- 更快收敛

### 主要渲染器

**CPU渲染器**：
- Arnold
- V-Ray
- RenderMan
- Corona

**GPU渲染器**：
- Octane
- Redshift
- Cycles（CPU/GPU）

## 技术对比

### 全局光照（GI）

**实时**：
- 光照贴图（预烘焙）
- 屏幕空间GI（近似）
- 探针系统
- Lumen（UE5）

**离线**：
- 完整路径追踪
- 光子映射
- 辐射度算法
- 多次反弹计算

### 阴影

**实时**：
- Shadow Mapping
- 级联阴影
- 体积阴影
- PCSS（软阴影近似）

**离线**：
- 光线追踪阴影
- 区域光软阴影
- 焦散
- 体积阴影

### 反射

**实时**：
- 反射探针
- 屏幕空间反射（SSR）
- 平面反射
- 光线追踪反射（RTX）

**离线**：
- 完整光线追踪
- 多次反弹
- 模糊反射
- 色散

### 材质

**实时**：
- PBR材质（近似）
- 有限的SSS
- 简化的透明度
- Shader优化

**离线**：
- 完整BSDF
- 精确SSS
- 复杂折射
- 多层材质

## 混合渲染

现代技术正在模糊两者界限。

### 实时光线追踪

**NVIDIA RTX / DirectX Raytracing**：
- 实时光线追踪反射
- 实时GI（RTGI）
- 光线追踪阴影
- 光线追踪环境光遮蔽

**性能要求**：
- 需要专用硬件（RTX GPU）
- 仍需要降噪
- 样本数较少

### Unreal Engine 5 新技术

**Lumen**：
- 动态全局光照
- 实时GI和反射
- 软件/硬件混合

**Nanite**：
- 虚拟几何体
- 电影级多边形数量
- 自动LOD

### 混合渲染管线

```
实时渲染 + 离线质量后处理
- 实时预览
- 离线最终渲染
- 同一场景资产
```

## 应用场景

### 游戏（实时）

```
- 角色：10K-50K多边形
- 纹理：2K-4K
- 光照：预烘焙+动态
- 目标：60 FPS
```

### 电影（离线）

```
- 角色：数百万多边形
- 纹理：8K-16K
- 光照：完整GI
- 时间：数小时/帧
```

### 建筑可视化（混合）

```
- 实时漫游预览
- 离线高质量出图
- VR体验
```

### 产品渲染（离线为主）

```
- 照片级质量
- 精确材质
- 完美光照
```

## 未来趋势

### 实时渲染方向

- 🚀 更强的光线追踪
- 🤖 AI降噪和超采样（DLSS）
- 🌍 更好的GI解决方案
- 📊 神经渲染技术

### 离线渲染方向

- ⚡ GPU加速普及
- 🌟 更快的收敛
- 💾 分布式渲染
- 🎯 AI辅助降噪（OptiX Denoiser）

### 融合趋势

- 实时质量不断提升
- 离线速度不断加快
- 工作流程统一
- 同一资产多用途

## 选择建议

### 选择实时渲染如果：

- ✅ 需要交互性
- ✅ 目标平台是游戏/VR
- ✅ 需要实时反馈
- ✅ 预算有限

### 选择离线渲染如果：

- ✅ 追求最高质量
- ✅ 时间不是限制
- ✅ 复杂光照场景
- ✅ 电影/广告项目
