## 什么是UV贴图

UV贴图是将3D模型的表面"展开"到2D平面的过程，以便应用纹理。U和V代表2D纹理空间的坐标轴（类似XY）。

## 为什么需要UV展开

- 📐 将纹理正确映射到3D模型
- 🎨 控制纹理在模型上的位置和比例
- 🔧 为手绘或程序生成纹理提供基础

## UV展开方法

### 1. 自动展开（Unwrap）

最常用的方法，基于模型的几何形状自动展开。

**Blender操作**：
```
1. 进入编辑模式（Tab）
2. 全选（A）
3. U键 → Unwrap
```

**优点**：快速、简单
**缺点**：可能产生变形，需要手动调整

### 2. 智能UV投影（Smart UV Project）

考虑到模型的角度和岛屿划分。

```
U键 → Smart UV Project
参数：
- Angle Limit: 角度限制（通常66度）
- Island Margin: 岛屿间距
```

### 3. 投影展开

#### 从视图投影（Project from View）
- 适用于平面物体
- 从当前视角投影

#### 立方体投影（Cube Projection）
- 适用于方形物体
- 从6个方向投影

#### 圆柱投影（Cylinder Projection）
- 适用于圆柱形物体
- 环绕展开

#### 球形投影（Sphere Projection）
- 适用于球形物体
- 两极会有挤压

### 4. 跟随活动四边面（Follow Active Quads）

基于活动面展开相邻面，适合创建规则的UV布局。

## 标记接缝（Seams）

接缝是UV展开时的"切割线"，决定了模型如何被展开。

### 标记接缝

```
1. 进入边选择模式（2键）
2. 选择要标记为接缝的边
3. Ctrl + E → Mark Seam
```

### 选择接缝的原则

✅ **好的接缝位置**：
- 隐藏的边（底部、背面）
- 自然的分界线（衣服的缝线）
- 硬边缘

❌ **避免的接缝位置**：
- 可见的平滑表面
- 需要连续纹理的区域

### 实例：人体模型

常见接缝位置：
- 身体内侧（腋下、腿内侧）
- 头后部中线
- 手掌内侧
- 脚底

## UV岛（UV Islands）

展开后形成的独立UV块称为UV岛。

### 岛屿管理

**选择UV岛**：
- L键（鼠标悬停在岛上）
- Ctrl + L（选中面上）

**打包UV岛**：
```
UV Editor中：
选中所有 → U键 → Pack Islands
参数：
- Margin: 间距
```

## UV编辑器操作

### 基本操作

```
选择：左键
框选：B键
移动：G键
旋转：R键
缩放：S键
钉住：P键（防止被修改）
```

### 对齐和排列

```
Ctrl + P: 固定选中的UV
Alt + P: 取消固定
Shift + W: 焊接/对齐UV
```

### UV网格可视化

- 边长度：显示拉伸情况
- 面积：显示缩放情况
- 角度：显示变形情况

## 纹理密度

保持一致的纹理密度很重要，确保纹理在模型不同部分的分辨率一致。

### 检查纹理密度

使用棋盘格纹理：
```
1. 添加材质
2. 使用Image Texture节点
3. 加载棋盘格图像
4. 检查方格大小是否一致
```

### 调整密度

- 大的UV岛 = 更高的纹理分辨率
- 小的UV岛 = 更低的纹理分辨率
- 重要部分可以占用更多UV空间

## 常见问题与解决

### 拉伸（Stretching）

**原因**：UV形状与3D几何不匹配

**解决**：
- 添加更多接缝
- 使用Pin + Unwrap调整
- 使用Minimize Stretch

### 重叠（Overlapping）

**原因**：多个面共享相同UV空间

**何时需要**：
- ✅ 对称模型（镜像UV）
- ✅ 重复元素（树叶、瓦片）

**何时避免**：
- ❌ 需要烘焙光照/AO
- ❌ 需要独特纹理

### UV超出0-1范围

**平铺纹理**：允许超出，纹理会重复
**唯一纹理**：应保持在0-1范围内

## UDIM工作流

UDIM允许使用多个纹理贴图（1001, 1002, 1003...），适合高精度模型。

### 使用场景

- 影视级角色
- 需要极高纹理细节
- 不同身体部位用不同贴图

### Blender中使用UDIM

```
1. UV Editor → Image → New → UDIM Tiles
2. 添加多个瓦片
3. UV可以分布在不同瓦片上
```

## 最佳实践

### UV展开流程

1. ✅ 分析模型，规划接缝
2. ✅ 标记接缝
3. ✅ 展开UV
4. ✅ 切割和调整岛屿
5. ✅ 检查拉伸和变形
6. ✅ 打包UV岛
7. ✅ 验证纹理密度

### 优化技巧

- 🔸 相似的部分可以重叠（节省纹理空间）
- 🔸 给重要区域分配更多UV空间
- 🔸 保持UV岛的方向一致（便于绘制）
- 🔸 留出足够的边距（避免纹理渗色）
- 🔸 0-1空间充分利用（提高纹理分辨率）

## 总结

良好的UV展开是高质量纹理的基础。花时间优化UV布局，可以在后续纹理绘制和渲染中获得更好的效果。
