# 3D动画的12条原则

## 动画12原则

迪士尼动画师总结的经典动画原则，同样适用于3D动画。

### 1. 挤压和拉伸（Squash and Stretch）

赋予物体重量和弹性感。

### 2. 预备动作（Anticipation）

动作前的准备，让观众预期即将发生的动作。

### 3. 演出布局（Staging）

清晰地呈现动作的意图。
