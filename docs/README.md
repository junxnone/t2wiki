# T2 Wiki

欢迎来到 T2 Wiki！这是一个基于 GitHub Issues 自动生成的知识库。

## 介绍

本 Wiki 通过 GitHub Issues 进行内容管理：

- 📝 创建 Issue = 创建文档
- ✏️编辑 Issue = 更新文档
- 🏷️ 添加标签 = 分类管理
- 🗑️ 关闭/删除 Issue = 归档/删除文档

所有文档会自动同步到此站点，实现零配置的知识管理。

## 如何使用

1. 在本仓库创建一个新 Issue
2. 填写标题和内容（支持 Markdown 格式）
3. 添加相应的标签进行分类
4. 等待几秒钟，文档将自动出现在侧边栏

## 快速导航

查看左侧边栏浏览所有文档，或使用顶部搜索框快速查找内容。
